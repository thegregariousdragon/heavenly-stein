# The Heavenly Stein

This package contains the active canon for **The Heavenly Stein**, a warm, absurd, continuity-heavy space-frontier saga centered on Heinz Schmitt and Der Himmelskrug.

## Continuity rebuild status

This package is a continuity-rebuild version.

The active canon keeps:

- character specifics and scene fingerprints;
- law, economy, corporations, institutions, and governance;
- colonies, stations, routes, jurisdictions, and media feeds;
- Der Himmelskrug’s redesigned station layout;
- ships, ship systems, transport, weapons, and force rules;
- clean-side/restricted-side routing, Arrival Control, Flight Control authority, cannabis/Grünraum rules, and station procedures;
- tested system precedents in [`canon/19-tested-story-precedents.md`](canon/19-tested-story-precedents.md).

The active canon does **not** keep old story plots, chronology, scene blocking, or one-off events unless a future canon update explicitly re-promotes them. No archived story prose is included in this package.

## Using this repo with AI

For human-facing instructions on using this repo with an AI assistant, start with [`ai-workflows/HUMAN_USER_GUIDE.md`](ai-workflows/HUMAN_USER_GUIDE.md).

Use [`ai-workflows/STORY_PIPELINE.md`](ai-workflows/STORY_PIPELINE.md) for story development, [`ai-workflows/CONTEXT_PACKS.md`](ai-workflows/CONTEXT_PACKS.md) to choose the smallest needed canon set, and [`planning/story-intake-template.md`](planning/story-intake-template.md) to turn messy ideas into draft-ready material.

Use [`ai-workflows/ASSISTANT_COLLABORATION_RULES.md`](ai-workflows/ASSISTANT_COLLABORATION_RULES.md) for decision-question, PR, conflict-check, merge, branch-cleanup, and local-artifact preferences.

Use [`ai-workflows/EPUB_PUBLISHING.md`](ai-workflows/EPUB_PUBLISHING.md) when generating local reader-facing EPUB files from publishable story Markdown.

Drafts, examples, and planning notes are not canon by default. New durable facts become canon only through [`ai-workflows/CANON_PROMOTION.md`](ai-workflows/CANON_PROMOTION.md).
