# The Heavenly Stein

This package contains the active canon for **The Heavenly Stein**, a warm, absurd, continuity-heavy space-frontier saga centered on Heinz Schmitt and Der Himmelskrug.

## Continuity rebuild status

This package is a continuity-rebuild version.

The active canon keeps:

- character specifics and scene fingerprints;
- law, economy, corporations, institutions, and governance;
- colonies, stations, routes, jurisdictions, and media feeds;
- Der Himmelskrug’s redesigned station layout;
- ships, ship systems, transport, weapons, and force rules;
- clean-side/restricted-side routing, Arrival Control, Flight Control authority, cannabis/Grünraum rules, and station procedures;
- tested system precedents in `canon/19-tested-story-precedents.md`.

The active canon does **not** keep old story plots, chronology, scene blocking, or one-off events unless a future canon update explicitly re-promotes them. No archived story prose is included in this package.
