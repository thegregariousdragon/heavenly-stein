# Quick Start

Use this file when starting a new writing or design session.

## Core premise

The Heinz Schmitt stories are a warm, absurd, continuity-heavy space-frontier saga about Heinz Schmitt and Der Himmelskrug.

The central mode is:

> Ridiculous premise, serious engineering underneath.

Der Himmelskrug, “The Heavenly Stein,” is a tall beer-stein-shaped orbital brewery/free-port/station. It is funny from a distance and serious up close: brewery, port, clean-side/restricted logistics facility, hospitality hub, residential community, learning center, public entertainment/culture venue, and route-safety institution.

## Sensory and accessibility note

For station scenes, check `canon/08-der-himmelskrug-station.md` and `quick-reference/STATION_BLOCKING_GUIDE.md` for level-specific surface, lighting, air/smell, and sound guidance. Use a few specific sensory details rather than long visual tours. Do not rely only on color; name the light mode, sign, district, door, or route.

## Continuity rebuild rule

Keep the world. Reset the old events.

Active canon includes:

- character specifics;
- law, economy, corporations, locations, media feeds, governance, and labor;
- ships, ship systems, transport, weapons, and force rules;
- Der Himmelskrug’s redesigned station layout;
- Arrival Control, clean-side/restricted-side routing, Flight Control authority, The Green Room / Grünraum, The Spare Tap, and station procedures;
- system precedents in `canon/19-tested-story-precedents.md`.

Old story plots, chronology, and scene blocking are not active canon. No old story archive is included.

## Crown fixed layout

The Crown is a four-quadrant civic cap:

- Command
- Administration / Civic
- Learning Center
- Museum

Command is strictly restricted. Other Crown quadrants are controlled by purpose.
