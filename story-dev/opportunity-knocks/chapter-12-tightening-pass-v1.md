# Opportunity Knocks — Chapter 12 Tightening Pass v1

Status: draft-only revision note.

This file does not create active canon or active chronology. It is a targeted correction note for `story-dev/opportunity-knocks/chapter-12-draft-v2.md`.

## Purpose

Chapter 12 Draft v2 uses **Mira Kade** as a story-local Race Ops coordinator after avoiding a collision with canon character Lian Zhou. The name **Mira** is too close to **Meera**, one of Oskar’s Garage Ops technicians already active in the Opportunity Knocks drafts.

To avoid reader confusion, replace **Mira Kade** with **Nessa Rook** in Chapter 12.

## Required replacements

In `chapter-12-draft-v2.md`, replace every Chapter 12 Race Ops coordinator reference as follows:

- `Mira Kade` → `Nessa Rook`
- `Mira` → `Nessa`, only where the reference is the Race Ops coordinator in Chapter 12

Do **not** change **Meera**, the Garage Ops technician.

## Intended corrected references

The corrected Race Ops coordinator references should read:

- `Behind her, Race Ops coordinator Nessa Rook reviewed entrant windows...`
- `“Garage Ops packet is clean,” Nessa said.`
- `Nessa allowed herself half a smile.`
- `Nessa looked over. “Should I log that?”`
- `Nessa typed.`
- `Nessa placed a race-eligibility slate on the table between them.`
- `Nessa began the review.`
- `Nessa’s hand paused above her slate.`
- `“Remote abort handshake,” Nessa said.`
- `“Race Ops display feed,” Nessa said.`
- `“Telemetry green,” Nessa Rook said from Race Ops.`

## Name-collision note

This pass preserves the previous Chapter 12 fix that avoids using canon character **Lian Zhou** as Race Ops coordinator. It further avoids a near-collision with Garage Ops technician **Meera** by using **Nessa Rook** instead.

## Canon status

**Nessa Rook** is a story-local Race Ops coordinator for this draft unless later promoted through the candidate-canon process.
