# Publishable Stories

This folder is for story packages intended for publication or publication review.

A publishable story package should include:

- a local `README.md` with title, status, and file map;
- the story prose file, when present;
- `canon-status.md` identifying which facts are already locked in canon and which details remain candidate-only;
- optional `publication-notes.md` for editorial, formatting, accessibility, or release notes.

## Canon relationship

Publishable prose can express canon, but canon remains governed by the files in `canon/`.

If a story introduces durable new facts, promote those facts through canon files rather than relying on prose alone.
